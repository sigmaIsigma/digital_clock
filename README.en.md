# Digital Clock & Calendar

A clean, minimalist digital clock and calendar web application designed to be used as a full-screen or windowed dashboard. Supports PWA for installation on mobile and desktop.

## Features

- **Dynamic Visuals**: Text color automatically changes between Day (White) and Night (Cornflower Blue) modes.
  - Supports **Auto**, **Manual (Day/Night Only)**, and **Custom Time** modes.
- **Customizable Display**:
  - 12h/24h time format.
  - Regular or Bold font weight.
  - Adjustable scale (10% to 300%).
  - Highlight style for "Today" (Round/Square), configurable separately for weekdays and weekends.
- **Interactive Calendar**: Automatically updates and adjusts layout for portrait and landscape orientations.
- **Audio Feedback**: Subtle beeps for seconds and top of the hour (can be muted).
- **PWA Support**: Installable as a standalone app on supported browsers.
- **Quick Return**: Tap or click 5 times rapidly (within 1.25 seconds) to return to the settings/start screen.

## Installation / Usage

Simply host the files on any static web server or open [this page](https://sigmaisigma.github.io/digital_clock/) in your browser.

1. Configure your preferences on the "CLICK TO START" screen.
2. Click the button to start the clock.
3. To change settings, tap/click anywhere on the screen 5 times quickly.

## Requirements

- A modern web browser with JavaScript enabled.
- (Optional) Service Worker support for PWA features.
