# デジタル時計 & カレンダー (Digital Clock & Calendar)

フルスクリーンでのダッシュボード表示を想定した、クリーンでミニマルなデザインのデジタル時計とカレンダーのウェブアプリケーションです。モバイルとデスクトップの両方でPWAとしてインストール可能です。

## 主な機能

- **動的なビジュアル**: 時間帯に応じてテキストカラーが自動的に切り替わります（昼：白色、夜：紺色）。
  - **オート**、**強制（昼のみ/夜のみ）**、<b>時間指定（カスタム）</b>の各モードをサポート。
- **カスタマイズ可能な表示**:
  - 12時間/24時間表示の切り替え
  - フォントの太さ（Regular / Bold）の切り替え
  - 全体の表示サイズ調整（10% 〜 300%）
- **レスポンシブなカレンダー**: 画面の向き（縦・横）に合わせてレイアウトが自動調整されます。
- **オーディオ・フィードバック**: 秒刻みや正時（0分）に控えめなビープ音を鳴らします（ミュート可能）。
- **PWA対応**: 対応ブラウザでスタンドアロンアプリとしてインストールできます。
- **クイック・リターン機能**: 画面を1.25秒以内に5回連続でタップ（またはクリック）すると、設定画面に戻ります。

## インストール / 使い方

静的ウェブサーバーにファイルを配置するか、ブラウザで直接 `index.html` を開くだけで使用できます。

1. 「CLICK TO START」画面で好みの設定を行います。
2. ボタンをクリックして時計を開始します。
3. 設定を変更したい場合は、画面上のどこでも良いので5回素早くタップ/クリックしてください。

## 動作要件

- JavaScriptが有効な最新のウェブブラウザ。
- （オプション）PWA機能を利用する場合は、Service Workerをサポートしているブラウザ。
